## 🔎 问题原因

### 1. 缓存数据过期

**文件**: `data/gold_price_latest.json`

**过期数据**:
```json
{
  "domestic_cny_per_gram": 1000.0,
  "source": "cngold_quote",
  "update_time": "2026-03-27 10:00",  // 4 天前的数据
  "fetch_time": "2026-03-27 10:00:02"
}
```

**问题**:
- 数据是 3 月 27 日的（过期 4 天）
- 缺少国际金价字段
- 系统使用默认计算（国内金价 * 7.2 / 31.1）

### 2. 验证器阈值错误

**文件**: `data_pipeline/validator.py`

**错误配置**:
```python
'gold_price_intl': (150, 350),  # ❌ 这是 20 年前的金价！
```

**问题**:
- 阈值设置基于 2000 年代初的金价（$200-350/盎司）
- 2026 年实际金价约 $2600-2700/盎司
- 导致异常数据通过验证

### 3. 数据源依赖单一

**问题**:
- 过度依赖本地缓存文件
- 实时获取功能未启用
- 备用源（浏览器）未正确使用
