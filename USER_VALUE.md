## 💰 核心价值

### 1. 每日投资简报 - 省时省力

**您得到的**:
- 📊 最新金价、宏观数据、市场情绪
- 🔮 次日价格预测（含置信度）
- 📈 3 张可视化图表
- ⚠️ 风险提示

**节省时间**: 从 2 小时研究 → 2 分钟阅读

**示例**:
```markdown
# 投资分析每日简报 2026-03-31

## 市场概览
- 国际金价：$2034.5/盎司 (+0.61%)
- 国内金价：¥486.2/克 (+0.43%)
- 美元指数：99.68 (-0.12%)
- VIX 恐慌指数：15.2 (低位)

## 价格预测
- 预测价格：¥492.5
- 方向：上涨 📈
- 置信度：中等 (65%)
- 信号：买入

## 主要依据
1. 技术面：RSI=58，MACD 金叉
2. 情绪面：新闻偏正面
3. 宏观面：美元走弱支撑金价

## 风险提示
⚠️ 预测仅供参考，不构成投资建议
```

### 2. 价格预测 - 决策参考

**您得到的**:
- 📍 具体价格区间（不是模糊的"可能涨"）
- 🎯 涨跌概率（不是"大概吧"）
- 📊 置信度评级（高/中/低）
- 🔍 预测依据（透明可查）

**预测能力**:
```
短期预测 (1-3 天): 目标准确率 > 60%
中期预测 (1-2 周): 目标准确率 > 55%
长期预测 (1 月+): 提供趋势方向
```

**实际效果**:
```
今日：金价 ¥486.2
预测：明日 ¥492.5 (上涨 +1.3%)
置信度：中等
结果：次日实际 ¥491.8 ✅ 方向正确
```

### 3. 多维度分析 - 专业视角

**您得到的**: 4 个维度专业分析

| 维度 | 分析内容 | 权重 |
|------|----------|------|
| **技术分析** | RSI、MACD、均线、布林带 | 25.5% |
| **情绪分析** | 新闻情感、市场热度 | 21.25% |
| **宏观分析** | 美元指数、VIX、油价、美债 | 21.25% |
| **动量分析** | 趋势强度、价格动能 | 17% |
| **时序模型** | 统计规律、移动平均 | 15% |

**对比散户**:
```
散户分析：看 K 线 + 听消息 + 凭感觉
系统分析：4 维度 32 指标 + 实时数据 + 量化模型
```

### 5. 预测验证 - 持续改进

**您得到的**:
- ✅ 每日预测 → 次日验证
- 📊 准确率统计（不隐瞒失误）
- 🔄 权重优化（根据验证结果调整）

**验证报告示例**:
```
过去 30 天预测验证:
- 总预测数：30 次
- 方向正确：19 次 (63.3%)
- 价格误差：<2%: 15 次，2-5%: 12 次，>5%: 3 次
- 平均误差：2.8%

结论：短期预测准确率达标 (>60%)
```

### 6. 风险预警 - 提前提醒

**您得到的**:
- 🚨 价格突破预警（>3% 波动）
- ⚠️ 低置信度预警（预测不可靠时）
- ❌ 系统错误通知（数据异常时）

**预警示例**:
```
⚠️ 低置信度预警

预测置信度：35% (低于 40% 阈值)
原因：
- 数据源质量下降 (C 级)
- 各因子分歧较大
- 宏观数据缺失

建议：谨慎参考，等待数据改善
```

### 7. 配置灵活 - 按需调整

**您能自定义**:
- 📊 预测权重（更相信技术 or 宏观？）
- ⏰ 缓存时间（更实时 or 更稳定？）
- 🎯 置信度阈值（更激进 or 更保守？）
- 📁 输出格式（Markdown / JSON / Excel）

**示例**:
```yaml
# 更保守的配置
prediction:
  confidence_thresholds:
    high: 0.85    # 提高标准
    medium: 0.65

# 更激进的配置
prediction:
  weights:
    technical: 0.35    # 更相信技术分析
    sentiment: 0.15    # 降低情绪权重
```

## 🎯 具体使用场景

### 场景 1: 考虑买入黄金

**传统做法**:
```
看新闻 → 听朋友说 → 看 K 线 → 凭感觉 → 买入
```

**使用系统**:
```
1. 查看每日简报 → 了解当前金价和趋势
2. 查看预测 → 明日预测上涨，置信度中等
3. 查看依据 → 技术面好 + 宏观面支撑
4. 查看风险 → 美元可能反弹
5. 决策 → 买入，但设置止损
```

**结果**: 更理性、更有依据的决策

---

### 场景 2: 持仓中，犹豫是否卖出

**传统做法**:
```
涨了怕跌 → 卖了又涨 → 后悔
跌了怕跌更多 → 割肉 → 反弹 → 后悔
```

**使用系统**:
```
1. 查看预测 → 预测继续上涨，置信度高
2. 查看依据 → 各因子一致看好
3. 查看历史 → 高置信度预测准确率 70%
4. 决策 → 继续持有
```

**结果**: 避免情绪化操作

---

### 场景 3: 想知道市场发生了什么

**传统做法**:
```
刷新闻 → 信息过载 → 看不懂 → 焦虑
```

**使用系统**:
```
1. 查看简报"市场概览" → 5 分钟了解全局
2. 查看"主要依据" → 理解驱动因素
3. 查看"风险提示" → 知道潜在风险
```

**结果**: 快速理解市场，不再焦虑

---

### 场景 4: 验证自己的判断
