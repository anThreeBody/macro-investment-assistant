```

    - 用当前金价验证准确性

    ↓

    - 多数据源对比（东方财富 + 金投网 + 新浪财经）
    - 置信度评估
    ↓
【步骤 2】生成每日简报
    - 保存今日预测到数据库
    - 计算最新准确率统计
    - 生成 Markdown 简报
    ↓
【步骤 3】输出简报
    - 保存到 daily_brief/
    - 显示文件路径
```
