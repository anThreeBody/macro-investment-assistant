## 📋 问题描述

### 问题 1: 宏观叙事框架未实现

**SKILL.md 描述的框架**:
```
政策/事件 → 经济影响 → 资产价格 → 投资策略
```

**实际系统实现**: 多因子加权打分模型

**缺失内容**:
- ❌ 没有政策→经济影响的因果推理
- ❌ 没有经济影响→资产价格的传导机制
- ❌ 没有资产价格→投资策略的映射关系

### 问题 2: 新闻来源单一

**现象**: 每日简报中的新闻全部来自「和讯财经」

**原因**:
1. `news_source.py` 中浏览器抓取只配置了东方财富和和讯财经
2. 腾讯新闻仅在注释中提到，没有实际实现
3. 百度搜索可能未正常工作

## ✅ 解决方案

### 方案 1: 实现宏观叙事分析器

**新文件**: `analyzers/macro_narrative.py`

**核心功能**:
1. **政策类型识别** - 6 大类政策
   - 货币政策（降准、降息、MLF 等）
   - 财政政策（基建、专项债、减税等）
   - 产业政策（新质生产力、科技创新、半导体等）
   - 监管政策（反垄断、合规、审查等）
   - 地缘政治（战争、冲突、制裁等）
   - 经济数据（GDP、CPI、PMI 等）

2. **叙事强度评估** - 3 个级别
   - 强（重磅、重大、首次、历史性）
   - 中（持续、进一步、加强）
   - 弱（提及、表示、关注）

3. **因果推理链**:
```python
政策识别 → 影响评估 → 资产含义 → 投资策略
```

4. **输出格式**:
   - 识别到的政策叙事（Top 10）
   - 资产含义推导（黄金、股票、债券、美元、原油）
   - 投资策略建议（整体策略 + 核心主题 + 风险提示）

**集成位置**: `presenters/brief_generator_enhanced.py`
