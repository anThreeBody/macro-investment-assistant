## 🚀 使用方法

### 方式 1: 运行每日简报
```bash
cd ~//workspaces/YOUR_WORKSPACEH/skills/Macro-Investment-Assistant
./run_daily.sh
cat daily_brief/brief_v8_20260407.md
```

### 方式 2: 独立使用恐慌贪婪指数
```python
from analyzers.fear_greed_index import FearGreedIndex

calculator = FearGreedIndex()
result = calculator.calculate_index(
    vix=15.2,
    equity_bond_spread=0.2,
    northbound_flow=35.8,
    volume=8500,
    sentiment=0.35
)
print(f"指数：{result['index_value']} - {result['signal']}")
```

### 方式 3: 独立使用事件日历
```python
from data_sources.event_calendar import EventCalendar

calendar = EventCalendar()
events = calendar.get_upcoming_events(days=7)
for event in events:
    print(f"{event.date}: {event.title}")
```
